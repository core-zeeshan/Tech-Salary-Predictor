# 💼 Tech Salary Predictor

An AI-powered salary prediction web application built using Machine Learning and Streamlit. The application predicts technology-sector salaries based on job title, education level, age, gender, and years of experience.

---

## 🚀 Features

✅ Machine Learning Salary Prediction

✅ User Authentication (Login & Signup)

✅ Prediction History Tracking

✅ Interactive Dashboard & Analytics

✅ Download Reports (CSV)

✅ Modern Responsive UI

✅ SQLite Database Integration

✅ Data Visualization with Plotly

---

## 🛠️ Technologies Used

- Python
- Streamlit
- NumPy
- Pandas
- Scikit-Learn
- Random Forest Regressor
- SQLite
- Plotly
- Pickle

---

## 🤖 Machine Learning Model

This project uses a **Random Forest Regressor** trained on technology job salary data.

### Input Features
- Job Title
- Education Level
- Gender
- Age
- Years of Experience

### Output
- Predicted Annual Salary

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/core-zeeshan/Tech-Salary-Predictor.git
```

### Move into Project Folder

```bash
cd Tech-Salary-Predictor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
